# 개인용 리뷰 도우미

팀 `pr-reviewer` 스킬의 P1~P5 리뷰에 더해, 개인 글로벌 스킬을 활용한 심층 분석을 수행한다.

## 추가 분석 항목

팀 리뷰 완료 후, 아래 개인 스킬을 활용해 추가 관점을 제공:

### 1. 코드 비교 (diff-review)
- 변경 전후를 라인별 대응 표 + 병렬 코드블록으로 시각화
- 에러 전파 경로 분석

### 2. 구현 대안 제시 (propose-fix)
- 리뷰 중 발견한 문제에 대해 다중 방안 비교
- 사이드이펙트 분석 + Before/After 코드

### 3. 코드 위치 추적 (find-in-code)
- 변경된 코드가 참조하는 곳, 영향받는 곳을 절대경로:라인번호로 추적
- 연결 흐름 시각화

### 4. 구조 건강도 (codebase-health)
- 변경이 전체 아키텍처에 미치는 영향 진단
- Shallow Module 후보, 커플링 핫스팟 식별

## 사용법

```
/my-review          ← 현재 브랜치의 변경사항 심층 리뷰
/my-review #123     ← PR #123 심층 리뷰
```

## 출력 형식

```
## 심층 리뷰

### 팀 리뷰 (P1~P5)
→ pr-reviewer 스킬 결과 참조

### 코드 비교
→ diff-review 형식

### 구현 대안 (문제 발견 시)
→ propose-fix 형식

### 영향 범위
→ find-in-code 추적 결과
```
